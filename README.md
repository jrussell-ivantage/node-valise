# load-conf

Easy access to your project's config files... no more require (
"../.../../my.conf" )

## Getting Started
(soon) Install the module with: `npm install load-conf`

```javascript
var load_conf = require( "load-conf" )
	, dbConf = load_conf( "db" )

// Do something with your db config
```
